# Images Folder

Place your images here. The website references the following images:

## Gallery Images
- `gallery-1.jpg` through `gallery-6.jpg` — Photos for the photo gallery section

## Memory Images
- `memory-1.jpg` through `memory-6.jpg` — Photos for the memory timeline section

## Recommended Sizes
- Gallery images: 400x400px or larger (square crop works best)
- Memory images: 600x400px or larger (landscape orientation)

All images should be `.jpg`, `.png`, or `.webp` format.
